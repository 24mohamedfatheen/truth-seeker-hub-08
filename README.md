Truth Lens – Fake News Detection Website

Project Overview
Truth Lens is an AI-powered website designed to help users detect and avoid misinformation online. By analyzing text, images, and videos, the system provides a trust score and highlights evidence of manipulation or bias. The project was developed as part of the Grade 12 Capstone Project (Session 2025–26, Subject Code 843) at Chennai Public School.

Features
Fake news detection tool (text, image, video input)
Trust score with explanation of results
Educational resources on misinformation and digital literacy
Verified news feed from reputable sources
